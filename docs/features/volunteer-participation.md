## 참여 흐름

참여 클릭 → 일정명·일시·장소 확인 modal → pending → API → 성공 안내 → 목록·상세·내 예정·요약 갱신.

## 취소 흐름

취소 클릭 → `봉사 시작 2시간 전까지만 직접 취소 가능` 안내 → API → 성공 안내 → 관련 query 갱신. 2시간 여부는 브라우저 계산이 아니라 `canCancel`과 서버 응답을 따른다.

## 오류 매핑

| 오류 | UI |
| --- | --- |
| 정원 마감 | 버튼 비활성화, 일정 재조회 |
| 시간 중복 | 겹치는 일정 안내 |
| 활동정지 | 참여 불가·관리자 문의 |
| 중복 신청 | 이미 신청 안내, 일정 재조회 |
| 취소 마감 | 취소 비활성화·관리자 문의 |

## 접근성·안전

modal은 제목 연결, 초기 focus, Escape, 닫은 뒤 trigger focus 복귀를 제공한다. mutation 중 중복 요청을 막고 실패 시 기존 화면 상태를 임의로 확정하지 않는다.

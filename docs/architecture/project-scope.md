## 목표

Next.js는 봉사단 사용자·관리자 화면, 입력 경험, API 호출, 상태 피드백, 접근성·반응형을 담당한다. Spring Boot는 권한, 상태 전이, 정원, 시간 충돌, 통계 등 최종 업무 판단을 담당한다.

## 포함 범위

- 사용자: 소개, 가입 신청·수정·상태, 일정 목록·상세, 참여·취소, 내 봉사활동
- 관리자: 신청, 단원, 일정, 출석 관리
- 공통: API Client, 폼·모달·토스트, loading/error/empty, Mock, 테스트

## 프론트 비책임

- 정원·시간 중복·취소 마감·권한·상태 전이의 최종 판단
- 일정 배열을 이용한 누적 실적 최종 계산
- DB 변경, 자동 마감·완료, 파일 진위와 악성 콘텐츠 최종 검증
- 관리자 메뉴 숨김만으로 보안을 구현하는 행위
